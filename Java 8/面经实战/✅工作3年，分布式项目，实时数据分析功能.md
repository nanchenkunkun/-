这里面候选人提到自己用了最大努力通知，但是实现方式竟然是自己在代码中sleep，然后自己重试，这个方案肯定是不过关的。因为sleep这种一旦应用重启之后，数据就都丢失了，任务就没人继续调度重试了。

所以好一点的方案肯定是用MQ，依赖他的可靠性来实现。
